## Description

DOD: As a developer design the navbar with the following functionality.

- A path or a breadcrumbs to navigate to the parent pages.
- A Tab switcher which should be able to render compoenent inside the viewport
- Avatar with user image and option to logout and user settings current.

Note:
1. All components should be modularized and reusable
2. Proper documentation string is mandatory
3. Initialy developer the UI with the hardcoded data but should match the current page's data structure
4. It should render the Navbar based on the page so configure it in such a way
